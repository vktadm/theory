## Битовые операции

#### not (~)
```python
a = 137
bin(a) # '0b10001001'

~a # -138
bin(~a) # '-0b10001010'

d = 0
bin(~d) # '-0b1'

b = -10
~b # 9
bin(~b) # '0b1001'
```

#### and (&)

```python
flags = 5
bin(flags) # '0b101'
mask = 4
bin(mask) # '0b100'

res = flags & mask # 4
bin(res) # '0b100'
```

#### or (|)

```python
a = 17
bin(a) # '0b10001'
b = 4
bin(b) # '0b100'

res = a | b # 21
bin(res) # '0b10101'
```

#### xor (^)

Применяется для шифрования данных.

```python
flags = 17
bin(flags) # '0b10001'
mask = 10
bin(mask) # '0b1010'

flags = flags ^ mask # 27
bin(flags) # '0b11011'

flags = flags ^ mask # 17
bin(flags) # '0b10001'
```

##### Приоритет: not -> and -> or + xor (одинаковый приоритет)

#### Побитовый сдвиг вправо (>>) и влево (<<)

```python
x = 160 # '0b10100000'
x = x >> 1 # '0b1010000' -> 80
# >> 2 - равноценно целочисленному делению на 2
x = x >> 2 # 20

x = x << 1 # 40
# << 2 - эквивалентно умножению на 2
```

## Модуль random
```python
import random as rd
# from random import randint

# генерирует случайное число от 0.0 до 1.0
n = 10
number = rd.random() * n

# генерирует случайное число в заданном диапазоне. Возвращает float
number = rd.uniform(1, 5)

# возвращает случайное число из определенного диапазона
number = rd.randint(20, 50)

# возвращает случайное число из определенного набора чисел
number = rd.randrange(10)  # значение от 0 до 10 не включая
number = rd.randrange(2, 10)  # значение в диапазоне 2, 3, 4, 5, 6, 7, 8, 9
number = random.randrange(2, 10, 2)  # значение в диапазоне 2, 4, 6, 8

numbers = [1, 2, 3, 4, 5, 6, 7, 8]

# перемешивает список
rd.shuffle(numbers)

# возвращает список из случайных элементов, 2-ой аргумент - кол-во элементов
lst = rd.sample(numbers, 5) # [4, 6, 8, 3, 5]

# возвращает случайный элемент списка
random_number = rd.choice(numbers)
```

Функции `random()`, `uniform()`, `randint()`, `randrange()` подчиняются равномерному закону распределения. Значения генерируются с равной вероятностью.
Гаусcовские величины - значения с нормальным распределением.

```python
import random as rd

rd.gauss(mu, sigma) # mu - математическое ожидание, sigma - среднеквадратическое отклонение

value = rd.gauss(0, 3.5)
```

#### .seed()

Результат генерации всегда будет одинаковым.

```python
import random as rd

rd.seed(1)
lst = [rd.randint(-5, 5) for _ in range(10)]

print(lst) # [-3, 4, -4, -1, -4, 2, 2, 2, 5, 1]
```
